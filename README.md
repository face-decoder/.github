Organisasi ini dibuat sebagai tempat untuk mengerjakan skripsi. Repositori-repositori yang ada di sini berisi kode dan dokumentasi yang berkaitan dengan penelitian dan pengembangan proyek skripsi.

## About This Organization

This organization was created as a dedicated space for working on an undergraduate thesis (skripsi). The repositories within contain code and documentation related to undergraduate thesis research and development.
